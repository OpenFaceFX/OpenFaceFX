# Pipeline & generation

The end-to-end orchestration plus the input stages that feed it: the naive
aligner and Montreal Forced Aligner parser, grapheme-to-phoneme conversion, the
ARPAbet phoneme inventory, the transcript-free energy fallback, and the prosody
tracker that turns pitch and loudness into typed events.
