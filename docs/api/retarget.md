# Retargeting

Remap the 15 visemes onto another rig's blendshape names and weights — ARKit,
Rhubarb, Preston-Blair, VRM, CC4, Ready Player Me, and more — via the built-in
presets or a custom mapping. See the [Retargeting guide](../retargeting.md) for
the preset tables and worked examples.
