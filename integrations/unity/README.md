# OpenFaceFX — Unity integration

[`com.openfacefx.runtime`](com.openfacefx.runtime/) is a Unity Package Manager (UPM) package that
plays OpenFaceFX facial-animation takes on Unity characters at runtime — streaming ARKit blendshape
weights (and optional head/eye bone pose) onto a `SkinnedMeshRenderer`.

**Install** (Package Manager ▸ *Add package from git URL*):

```
https://github.com/OpenFaceFX/OpenFaceFX.git?path=integrations/unity/com.openfacefx.runtime
```

See the package [README](com.openfacefx.runtime/README.md) for a 60-second quick start and the
**Rocketbox ARKit** sample. Requires Unity 2021.3+. MIT-licensed.

> Built and format-verified against real OpenFaceFX exports; compile/run it in the Unity Editor to
> confirm against your specific rig.
